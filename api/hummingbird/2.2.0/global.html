search(query, callback, optsopt)

Finds matching names and returns them in order of best match.

Parameters:
Name Type Attributes Description
query string

the string against which matches will be returned
callback function

a function that takes the resultSet array and a profile object as arguments
opts object <optional>

an options object with properties that influence the results properties

Properties
Name Type Default Description
howMany int 10

the maximum number of results to be returned
startPos int 0

how far into the sorted matched set should the returned resultset start
scoreThreshold float 0.5

(number between 0,1 inclusive) only matches with a score equal to or greater than this fraction of the maximum theoretical score will be returned in the result set
boostPrefix boolean true

whether an additional boost applies to results that start with the first query token
secondarySortField string 'name'

if provided, results are sorted first by score descending, then by the property represented by this string
secondarySortOrder string 'asc'

('asc' or 'desc') optionally specifies whether sort on secondarySortField is ascending or descending
